# Manage Products Apps

this apps is crud operation to create , update , delete and show products, made with :
- vue js
- vuetify
- html
- json-server


## Requirement
- npm version 6.14.4

## How To Run
- first install json server with npm

```bash
npm i json-server
```

- then run project with
```bash
npx json-server --watch db.json
```
- open index.html
